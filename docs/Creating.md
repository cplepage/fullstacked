# Creating

### Command
It generates the default starter files, simply run the following command:
```shell
npm init fullstacked
```
This creates the starter files :
```text
├── server
│   └── index.ts
├── tests
│   ├── e2e.test.ts
│   └── integration.test.ts
└── webapp
    ├── index.css
    ├── index.html
    └── index.ts

```
These files are your entrypoint for your new web app.
